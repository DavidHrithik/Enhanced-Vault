# Test Management Tool Frontend (React)

This is the frontend for the Test Management Tool. It uses React, Tailwind CSS, Framer Motion, and Axios.

## Setup
1. Ensure you have Node.js 18+ and npm installed.
2. Install dependencies: `npm install`
3. Start dev server: `npm start`

## Features
- Landing page with animated tiles
- Test Accounts and Datasets management
- Modals, search, filter, upload
- Responsive, modern UI
